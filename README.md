# rua 

A toy Lua compiler in Rust.

---

## How to play
1. Install Lua v5.1 or later, and make sure `lua` is in your `PATH`.
2. Run `cargo build` 
3. Use `rua input.lua -o input.luac` to translate Lua source file into bytecode.
